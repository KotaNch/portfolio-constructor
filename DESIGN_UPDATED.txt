ВКЛАДКА "АДМИНИСТРАТОР":
  • Сканирование FTC портфолио

  ✓ Функции скачивания PDF
  ✓ Поиск по шаблонам (TF-IDF)
  ✓ Фильтрация по типам
  ✓ Просмотр информации

  ✓ Обработка ошибок

API (backend/app.py):
  ✓ GET /api/download/<id> - скачивание портфолио
  
API (portfolio_templates_generator/api.py):
  ✓ POST /api/search - поиск по промту
  ✓ GET /api/download/<id> - скачивание шаблона
  ✓ GET /api/random - случайные шаблоны
  ✓ GET /api/metadata/<id> - информация о шаблоне

💡 СОВЕТЫ:

1. Для первого запуска сгенерируйте 100 шаблонов:
   - На вкладке "Администратор"
   - Введите "100"
   - Нажмите "Сгенерировать"

2. После генерации откройте вкладку "Шаблоны"
   - Там будут готовые PDF файлы

3. Используйте поиск для нахождения нужных:
   - "синий минимализм"
   - "корпоративный"
   - "творческий"

❓ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:

1. Backend не запускается?
   python -m pip install -r backend/requirements.txt

2. Генератор не работает?
   python -m pip install -r portfolio_templates_generator/requirements.txt

3. Браузер не открывает страницу?
   Откройте вручную: http://localhost:5000

4. Ошибка "портал занят"?
   Закройте другие приложения на этом портале
